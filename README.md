# dphil-thesis
This repository contains the code to generate and simulate the models described in Tarit Konuntakiet's DPhil thesis, completed at the University of Oxford, UK in 2021. A full copy of the thesis can be obtained from http://ora.ox.ac.uk/. Two different whole-plant metabolic models of growth are presented in the thesis, FM-SoFBA-SiFBA and WholePlantMetabolism.
